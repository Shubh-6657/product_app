# Product App

## Description
A React/Next.js app that fetches and displays products from a mock API. Features include filtering, sorting, and a product details modal.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the app: `npm run dev`.

## Features
- Fetch and display products.
- Filter by category.
- Sort by price.
- Responsive UI.
- Product details modal.
